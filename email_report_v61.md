## Módulo de Calidad Visual BakeOS
Se ha transformado el dashboard de calidad en un módulo interactivo de análisis en tiempo real, optimizado para el uso en planta y obrador.

### Funcionalidades del Nuevo Interfaz:
1.  **Visor Inmersivo**: Cámara a pantalla completa con guías AR inteligentes que cambian de color (Blanco/Naranja/Verde) según el estado de detección del objeto.
2.  **Carrusel de Selección**: Selector inferior traslúcido para alternar rápidamente entre categorías (Cookies, Ensaladas, Focaccias, Catering).
3.  **Panel de Análisis Dinámico**: Despliegue tipo "Slide-up" tras la captura, mostrando:
    - **Métricas IA**: Indicadores circulares para Presencia, Estética y Frescura.
    - **Score Global (1-5)**: Calificación inmediata del lote.
    - **Diagnóstico de Defectos**: Resaltado automático de fallos detectados (ej. Diámetro insuficiente).
4.  **Flujo D1**: Botón de validación directa para subir el registro a la base de datos D1 tras la aprobación del operario.
5.  **Galería de Auditoría**: Acceso rápido al histórico de fotos analizadas para comparación estética.
